Home

This is the home page for FaRCX, which is software for controlling LEGO® Mindstorms RCX robots over the internet. The robot which is controlled is a small "Rover" on a simulated Martian surface. Using the links above, you can drive the Rover, learn about FaRCX, follow FaRCX on Twitter, and view the FaRCX source code on GitHub.
